---
title: Authentication Best Practices
---

## Security

1. Change default credentials immediately
2. Use strong passwords
3. Regularly rotate API tokens
4. Set appropriate token expiration dates

## Device Management

1. Use meaningful device names
2. Tag devices appropriately
3. Flag devices for maintenance when needed
4. Regularly review team assignments

## Node Configuration

1. Store node credentials securely
2. Use environment variables for sensitive data
3. Monitor node connection status
4. Implement proper error handling for authentication failures
